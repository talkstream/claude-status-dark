How It Works

1
Install a userscript manager for your browser
Safari → Userscripts (free, App Store) · Chrome/Firefox → Tampermonkey
2
Click the Install Dark Theme button above
Your userscript manager will ask you to confirm
3
Open status.claude.com — it's dark now!
Works automatically every time you visit
