What does this actually do?
Replaces the bright white theme on status.claude.com with a warm dark palette. Same info, much easier on the eyes — especially at 2 AM when you're wondering why Claude went quiet mid-conversation.
Is it safe?
100% open source. It's a CSS theme with a small script that handles dynamic elements. No tracking, no data collection, no network requests. Read every line yourself.
Does it work on my browser?
Safari (via Userscripts app), Chrome, Edge, Brave (via Tampermonkey), and Firefox (via Tampermonkey). Basically everything except Internet Explorer, but if you're using IE, dark mode is the least of your problems.
Will it break the status page?
Nope. It only changes colors. All status information, links, and functionality remain exactly the same. If anything breaks, just disable the script.
